El modelo relacional

A través de las siguientes páginas vamos a aprender los conceptos más importantes sobre el modelo relacional. 

El modelo relacional se trata de una forma de organizar datos basada en conceptos matemáticos como la lógica de predicados y la teoría de conjuntos. 

Prerrequisitos: para estudiar los conceptos de las siguientes páginas, se entiende ya estudiado el modelo entidad-relación (ER).
